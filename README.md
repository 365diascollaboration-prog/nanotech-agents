## ⚡ Quick Start

### Install to Claude Code

```bash
# Clone the repo
git clone https://github.com/365diascollaboration-prog/nanotech-agents.git

# Copy agents to Claude Code
cp nanotech-agents/agents/*.md ~/.claude/agents/
```

### Activate an agent in Claude Code

```
Use the video-producer-latam agent to help me produce my next episode.
```
